# Main Service

SmartCast Robotics main service manages task lifecycle, equipment assignment,
state updates, event routing, traffic control, and monitoring data.

## Module Layout

```text
src/main_service/
├── main.py
├── task_manager.py
├── task_allocator.py
├── task_executor.py
├── robot_adapter.py
├── state_manager.py
├── event_bridge.py
├── traffic_manager.py
└── monitor_agent.py
```

## Test Layout

```text
tests/
├── unit/
├── integration/
└── conftest.py
```
